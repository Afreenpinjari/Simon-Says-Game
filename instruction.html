How to Play Simon Says

Simon Says is a fun memory game. Follow the steps below to play:

  1. Press the "Start" button on the main game page.
  2. Watch the sequence of colors
  3. Repeat the sequence by clicking the colored buttons in the correct order.
  4. After each correct round, the sequence will get longer.
  5. If you click the wrong color, the game will end.
  6. Try to remember as long a sequence as possible to get the highest score!
